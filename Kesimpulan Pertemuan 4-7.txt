Kesimpulan Pertemuan 4-7
Deskripsi :
- Dataset kelulusan_mahasiswa.csv (jumlah baris dan variabel mahasiswa dan kelulusan) tersedia di repo.
- Notebook melakukan proses dari eksplorasi data (EDA), pembersihan / preprocessing, pemodelan klasifikasi untuk prediksi kelulusan mahasiswa.
- Output: model akhir (termasuk model.pkl) dan grafik learning curve (learning_curve.png). 
GitHub
Ringkasan :
- Fokus pertemuan ini adalah memahami bagaimana data mahasiswa (fitur seperti IPK, kehadiran, dsb) bisa dipakai untuk memprediksi kelulusan atau tidak.
- Pekerjaan mencakup seluruh alur: load data → bersihkan → pelajari fitur → latihan model → evaluasi.
- Model dan artefak (model.pkl) menunjukkan bahwa materi sudah sampai ke tahap “siap deploy” atau “siap prediksi”.

Kesimpulan Pertemuan 6
Deskripsi :
- Notebook ini menunjukkan sesi yang lebih lanjut—kemungkinan implementasi model lanjutan seperti ensemble, ataupun deployment awal.
- File rf_model.pkl (random forest model) dan grafik roc_test.png menunjukkan bahwa di pertemuan ini dicapai model ensemble dengan evaluasi ROC.
- Ada juga file processed_kelulusan.csv yang menunjukkan preprocessing dan persiapan data sudah matang sebelum modeling.
Ringkasan :
- Pada pertemuan ini mahasiswa belajar bagaimana menggunakan data yang sudah diproses untuk model yang lebih kompleks (misalnya Random Forest) dan mengevaluasi dengan metrik yang lebih tajam (ROC).
- Ini meningkatkan kualitas prediksi dan membawa proyek mendekati tahap industrialisasi/model siap pakai.
- Pemisahan model dan data preprocessing menunjukkan alur yang lebih profesional (data pipeline → modeling → evaluasi).
